AI-powered itineraries that help travelers make the most of every layover — from 2 hours to a full day.

🤖 Multi-Agent AI 🗺️ Google Maps API ⚡ Real-time Streaming 🎭 5 Traveler Personas
The Problem

Layovers are wasted time

Millions of travelers sit in airports not knowing they could be exploring a new city — safely and efficiently.

😰

Analysis Paralysis

Researching what to do, how to get there, and whether you'll make it back in time is overwhelming and stressful.

⏳

Time Uncertainty

Travelers don't know how to account for transit time, departure buffer, and activity duration all at once.

🌍

No Personalization

Generic travel guides aren't tailored to your specific interests, budget, or the few hours you actually have.

The Solution

One input. A complete itinerary.

Layover Planner combines Claude AI agents with real-time Google Maps data to generate a safe, personalized city plan in seconds.

🤖

Multi-Agent Orchestration

A parent orchestrator delegates to specialist persona agents — each using live Maps data to surface the best options.

⏱️

Smart Time Budgeting

Automatically applies a 90-minute departure buffer and calculates round-trip transit time for every activity.

🗺️

Real Places, Real Hours

Searches live Google Maps data to verify venues are open, within budget, and reachable in your time window.

🎭

Persona-Based Planning

Choose from Foodie, Hiker, Culture, Shopper, or Relaxer — or mix multiple interests for a blended itinerary.

AI Agent Design

Orchestrator + Persona Specialists

🧠 Parent Agent

Orchestrator

Receives the full user context and decides which persona tools to invoke. After collecting all specialist recommendations, it synthesizes a conflict-free, time-ordered JSON itinerary.

  • Up to 20 agentic turns
  • Handles persona tool calls
  • Enforces time & budget rules
  • Outputs structured Pydantic model
🔧 Sub-Agents

Persona Specialists

Each specialist receives a focused prompt and accesses 4 Maps MCP tools to find real, open, budget-appropriate venues near the airport.

  • Up to 15 agentic turns each
  • Search → Verify → Filter → Rank
  • Return structured JSON arrays
